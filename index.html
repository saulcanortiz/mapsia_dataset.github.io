  • Welcome to Mosquito dataset

    Mosquito dataset is an open-source pavement distress bank of images collected with a low-cost and high-definition vehicle-mounted system for Pavement Distress Detection and Segmentation through Deep Learning algorithms.

    The data acquisition system is a drone mounted on a 3D-printed car mount with a suction system attached to a vehicle. This dataset was collected within MAPSIA project (UC), from IFCA and GITECO researchers.
